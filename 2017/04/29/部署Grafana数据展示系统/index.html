Grafana 是 Graphite 和 InfluxDB 仪表盘和图形编辑器。
Grafana 是开源的，功能齐全的度量仪表盘和图形编辑器，支持 Graphite，InfluxDB 和 OpenTSDB。
Grafana 主要特性：灵活丰富的图形化选项；可以混合多种风格；支持白天和夜间模式；多个数据源；Graphite 和 InfluxDB 查询编辑器等等。

认识Grafana

Graphite 指标编辑器

  • Graphite 指标表达解析器
  • 功能齐全的查询功能
  • 快速添加和编辑函数和参数
  • 模板化查询
  • See it in action

图形化

  • 快速渲染，甚至是较大的时间跨度
  • 点击和拖拽缩放
  • 多个 Y 轴
  • 条形，折线，点
  • 智能 Y 轴格式化
  • 系列切换和颜色选择
  • Legend values 和格式化选项
  • 网格阈值，轴标签
  • Annotations

仪表盘

  • 创建，编辑，保存和搜索仪表盘
  • 修改列宽和行高
  • 拖拽面板重新编排
  • 使用 InfluxDB 或者 Elasticsearch 作为仪表盘存储
  • 导入和导出仪表盘（JSON 文件）
  • 从 Graphite 导入仪表盘
  • 模板
  • Scripted dashboards
  • Dashboard playlists
  • 时间范围控制

InfluxDB

  • 使用 InfluxDB 作为指标数据源，注释源和仪表盘存储
  • 查询编辑器

OpenTSDB

  • 作为指标数据源
  • 查询编辑器
