playbooks 是一种简单的配置管理系统与多机器部署系统的基础。与现有的其他系统有不同之处,且非常适合于复杂应用的批量部署
Playbooks 的格式是YAML，语法做到最小化

这里讲述的是使用ansible-playbooks的roles安装LNMP应用

环境：
　　　ansible 2.2.1.0
　　　Centos 6.6
　　　PHP 7.0.16 (fpm-fcgi)
　　　Nginx 1.11.7
　　　Mysql 5.5.54

注：之前可能是在最后安装的mysql，把之前在PHP上安装的一个插件(php70w-mysql)弄丢了，所以就把mysql放最前面安装了
